# Pulsyon

Pulsyon is an API health and observability platform being built to monitor synthetic service traffic, ingest log events, and surface backend reliability metrics such as latency, error rate, and uptime. 

## Goal
Build a production-style backend system that:
- ingests high-volume API log events
- stores and analyzes service health data
- computes latency percentiles and error metrics
- explores backend performance, scalability, and observability concepts

## Planned Stack
- Node.js
- Express
- PostgreSQL
- Redis Streams
- Redis Cache
- Docker
- k6
- GCP Cloud Run
- GitHub Actions

## Planned Features
- Event ingestion pipeline
- Analytics APIs for p50/p95/p99 latency
- Error rate and uptime monitoring
- JWT authentication and role-based access control
- Redis caching and rate limiting
- Load testing and performance benchmarking
- Cloud deployment with CI/CD
